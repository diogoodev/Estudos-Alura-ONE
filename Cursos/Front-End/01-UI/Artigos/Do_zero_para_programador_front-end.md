# Do zero para programador front-end

Um duvida muito comum quando inciamos a nossa buscar para tornamos programadores front-end é por de começar, existem muitas maneiras, aqui vamos ver uma delas.

## Passo 1: HTML E CSS

O HTML é uma linguagem de marcação, responsável por dá estrutura as paginas Web
O CSS é o responsável por dar estilo as paginas web.
Entender como eles funcionam e praticar bastante é o básico a se fazer aqui, é preciso ir mais afundo e entender bem sobre responsividade web e como criar sites que funcionem bem tanto no mobile como no desktop.

## Passo2: Logica de programação e Javascript

Depois de aprender o html e o css podemos seguir agora com a logica de programação, essa parte é especialmente importante pois irá te preparar para entender a logica de uma linguagem de programação no caso javascript, mas você verá que apos aprender logica você poderá aplicar a qualquer outra linguagem. Durante essa jornada você verá (como declarar variáveis, loops, funções...). Apó isso estará completa a tríade clássica do desenvolvedor front-end.

## Passo 3: Bibliotecas

O mercado de trabalho utiliza muito bibliotecas e frameworks e por isso é importante ter o conhecimento delas, além disso elas facilitam a vida do programador através da padronização.
Alguns frameworks famosos são bootstrap, react, vue. Todos eles são ferramentas poderosíssimas nas construção de aplicações web.
O jQuery é exemplo de biblioteca, com ela conseguimos realizar modificações mais facilmente na pagina.

## Passo 4: Mais JavaScript

Nessa passo o foco é ir mais afundo na linguagem javascript entender as novidades da versão mais recente e aplicar em projetos.

## Passo 5: Frameworks

Aqui o foco é entender as Single Page Applications, e como os frameworks facilita as criações delas. React e angular são extramente populares e podem ser o seu foco nessa etapa.

## Passo 6: Pré-processador

Aprender a utilizar pré-processadores CSS como SASS. Aqui a ideia é entendo como podemos deixar o css mais poderoso, pois os pre-processadores são compiladores que traduzem uma sintaxe rebuscada que pode trabalhar com funções e variáveis no css.

## Passo 7: Noção de como funciona um back-end

Ter uma noção do funcionamento do backend, facilita a vida do desenvolvedor frontend e entender como a integração do back e do front ocorre é extramente util em um time de desenvolvimento.
Aqui você pode escolher uma linguagem para ver como o php, python ou java.
